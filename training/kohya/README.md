# Kohya sd-scripts Training

Use Kohya/sd-scripts on RunPod for production FLUX LoRA training. The configs in `training/configs/` define the first experiment targets:

- `bk_bini_body_v1_conservative`
- `bk_bini_body_v1_standard`
- `bk_bini_braces_v1`

ComfyUI should not be the production trainer. Use it for generation, evaluation, and review only.
